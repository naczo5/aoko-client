title: Chest ESP
description: Highlights nearby chests through walls.
---

*Chest ESP* highlights chests through walls so you can spot loot containers from a distance.

## Version support

1.8.9 · 1.21.x · 26.1

## Settings

| Setting | Description | Range / Default |
| ------- | ----------- | --------------- |
| Max count | Maximum number of chests highlighted at once (nearest first). | `1`–`20` / `5` |

## Usage notes

- Chests are drawn as overlays through the bridge and remain visible through terrain.
- **Max count** keeps the overlay readable by only highlighting the nearest chests and culling the rest by distance.

:::tip
Lower the **Max count** in chest-dense areas (like loot rooms) to avoid a cluttered screen.
:::
